Put Blender-authored VFX GLBs here.

Recommended examples:
- HitSpark.glb        : small crossed flash mesh or burst card mesh
- Slash_WideArc.glb   : curved slash arc mesh
- Shockwave_Ring.glb  : flat ring mesh for ground impact

AXVFXKit does not require AXEngine to procedurally create the effect shape.
The artist creates the visual mesh in Blender; AXEngine controls lifetime,
scale/alpha curves, transform, tint, socket attachment intent, and event spawning.
